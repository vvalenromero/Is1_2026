# IS2
Ingeniería de Software II - Repositorio de Cátedra
Este repositorio contiene los trabajos prácticos, proyectos y documentación desarrollados para la asignatura Ingeniería de Software II, correspondiente a la Licenciatura en Sistemas de la Universidad Autónoma de Entre Ríos (UADER), Facultad de Ciencia y Tecnología (FCyT).

🎯 Objetivo de la Materia
El enfoque de este espacio es profundizar en el ciclo de vida de desarrollo de software, haciendo especial énfasis en la gestión de proyectos, aseguramiento de la calidad (QA), metodologías ágiles y el diseño de arquitecturas robustas.
